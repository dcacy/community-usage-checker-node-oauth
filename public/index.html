Welcome to the IBM Connections Cloud Community Checker! This application will list the communities you can access, and show you some details about those communities. It uses OAuth, so clicking the Login button below will attempt to log you in to IBM Connections Cloud. If that succeeds, you'll be prompted to grant access to this application, after which it will show you your communities.
